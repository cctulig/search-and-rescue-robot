void PIDMotor::startInterpolation ( float  newSetpoint,
long  msTimeDuration,
interpolateMode  mode 
)

Start a linear interpolation of the motor

The position setpoint will follow a trajectory over time the setpoint will start where it currently is, and arrive at the desired location after the number of msTimeDuration MS have elapsed. The trajectory can be either linear, with the LIN mode or the trajectory can be either Sinusoidal, with the SIN mode the units for the variable is in sensor 'ticks'

Parameters
newSetpointthe new setpoint in motor ticks
msTimeDurationthe time of translation in miliseconds
modeThe type of interpolation LIN or SIN
Note
To get immediate action set msTimeDuration to 0
